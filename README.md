🌮 Taco Shop - HTML Practice Project
Overview
This is a simple HTML-only project designed to practice basic HTML concepts. It's a small website for a fictional taco shop with sections for About, Menu, Contact, and Store Hours. This project is purely static, showcasing essential HTML tags and structure without any additional CSS or JavaScript.

Project Structure
The website consists of a single HTML file that includes the following sections:

About - A brief introduction to the taco shop.
Menu - Lists different types of tacos with a description.
Store Hours - Displays the hours of operation.
Contact - Provides contact information, including phone number and address.
Features
Basic HTML Layout: Uses headings, paragraphs, lists, and links.
Navigation: Includes a simple navigation bar to access different sections.
Accessibility: Uses semantic tags for better structure and readability.
